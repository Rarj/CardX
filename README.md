# Usage
You can implement with a straightforward by typing `X(...)`
```
  X(
    titleCard = "Paypal",
    iconCard = R.drawable.ic_visa,
    numberCard = "1234 1234 1234 1234",
    nameCard = "Big Smoke",
    validDate = "01/28",
    cvvNumber = "555"
  )
```
